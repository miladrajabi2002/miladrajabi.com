# miladrajabi.com

رزومه تک‌صفحه‌ای میلاد رجبی با طراحی مدرن (RTL) بر پایه `PHP`، `HTML`، `CSS` و `JavaScript`.

## اجرا در لوکال

```bash
php -S localhost:8080
```

سپس در مرورگر باز کنید:

- `http://localhost:8080`

## ساختار فایل‌ها

- `index.php` — ساختار صفحه رزومه
- `styles.css` — استایل و طراحی مدرن/متریال
- `script.js` — انیمیشن لود و اسکرول

## شخصی‌سازی

- متن‌ها و بخش‌ها داخل `index.php` قابل ویرایش هستند.
- رنگ‌ها و فاصله‌ها در `:root` فایل `styles.css` تنظیم می‌شوند.
- رفتار انیمیشن‌ها در `script.js` قابل تغییر است.
